# Design Brief

## Preferred style

- Modern dark theme with clean contrast.
- Professional enough for recruiters and hiring managers.
- Should feel technical, calm, and credible rather than flashy.
- Suggested palette: near-black background, slate panels, white text, muted gray body text, and one accent color such as cyan/blue/teal.
- Use cards, clean dividers, and a simple timeline.

## UX priorities

- Recruiter should understand the profile in the first 10 seconds.
- Resume download buttons should be visible in the hero and resume section.
- Contact details should be easy to find.
- Skills should be grouped and scannable.
- Projects should emphasize what was built, what tools were used, and why it mattered.

## Accessibility

- Use semantic headings.
- Maintain readable color contrast.
- Make buttons keyboard accessible.
- Avoid tiny font sizes.
- Make the site work well on mobile.
